
## 0 进度计划 课程结构    
老男孩原计划一周一天的课程，那么就是一周8小时。
既然如此，设置为一天在Python上面的学习，应该为一天2-3小时。3-4天完成一周的课程。

可以这么考虑，2小时（视频时间，不是实际时间）听。2小时的练习。
